# NutriTrack Mobile App

NutriTrack is a mobile application designed to help users track their daily nutrition and dietary habits.

## Features
- User authentication
- Meal logging and nutrition tracking
- Daily nutrition summaries
- REST API integration

## Tech Stack
- Frontend: React Native
- Backend: Node.js / Django
- Database: PostgreSQL
- Version Control: Git & GitHub

## Purpose
This project demonstrates full-stack mobile application development, REST API integration, and clean UI design.
